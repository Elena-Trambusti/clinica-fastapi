# 🏥 Gestionale Clinica Pro

Questo è un software gestionale completo per una clinica medica, progettato per gestire medici e turni in modo sicuro e professionale.

## 🚀 Caratteristiche principali
- **Backend**: Realizzato con **FastAPI** (Python) e database **SQLAlchemy**.
- **Sicurezza**: Sistema di autenticazione con **Token JWT** e hashing delle password.
- **Frontend**: Interfaccia moderna in HTML5 e Bootstrap con tema celeste.
- **Database**: Gestione relazionale tra Medici e Turni.

## 🛠️ Come testarlo
1. Registrati tramite l'endpoint `/register`.
2. Effettua il login dalla pagina principale.
3. Gestisci il database dei medici e assegna i turni nelle stanze disponibili.
